# Submission Checklist

- [x] Memory simulator implemented (metadata-only, byte-addressable)
- [x] First Fit / Best Fit / Worst Fit allocators
- [x] Buddy allocator implemented separately
- [x] Block splitting and coalescing
- [x] Memory statistics and fragmentation metrics
- [x] Multilevel cache simulation (L1 + L2)
- [x] Interactive CLI / REPL
- [x] Design document included
- [x] Test workloads included
- [x] Demonstration logs included
- [x] Builds with GNU Make (C++17)
